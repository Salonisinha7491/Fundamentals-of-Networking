Glossary

Term Description

Chapter 1 : computer Network

Network A network is a collection of computers, servers, mainframes, network devices, peripherals, or other devices connected to allow data sharing.
Switch A switch operates in the layer 2, i.e. data link layer of the OSI Model.
It is an intelligent network device that can be conceived as a multiport network bridge.
It uses MAC addresses (addresses of medium access control sublayer) to send data packets to selected destination ports.
Hub A hub operates in the physical layer of the OSI model.
A hub cannot filter data. It is a non-intelligent network device that sends message to all ports.
It primarily broadcasts messages. So, the collision domain of all nodes connected through the hub stays one.
Bandwidth Bandwidth is a measure of the data transfer rate or capacity of a given network. It is a crucial network measurement for understanding the speed and quality of a network.
Network bandwidth is commonly measured in bits per second (bps).
Throughput Throughput is the actual data transferred successfully in a network.
Practically, the network throughput would always be less than the network bandwidth.
Latency Network latency is a measure of how long it takes a message to travel from one device to another across a network.
The more delays there are, the longer it takes to transmit data across a network.
Latency is affected by the number of devices on the network and the type of connection device.

chapter 2 : OSI Model

Dialog control It allows the systems to communicate in either half-duplex or full-duplex mode of communication.
Segmentation Segmentation means to divide something into pieces. When data arrives at the transport layer from the upper layers, it is taken then divided into segments. That is why data at this layer is called segments rather than data. Thus, The PDU (Protocol Data Unit) of this layer is segment(s).
Packetization The transport layer creates packet out of the messages received from the application layer. Packetizing is a process if dividing a long message into smaller ones. These packets are then encapsulated into data field of the transport layer packet and headers are added .
ARP The Address Resolution Protocol is a layer 2 protocol used to map MAC addresses to IP addresses.
RARP RARP is used to convert the Ethernet address to an IP address. It is available for the LAN technologies like FDDI, token ring LANs, etc.

chapter 3 : IEEE 802
DA Destination Addresses : It contain the address of the destination node within a network.
SA Sourece Address : It contain the address of the source node within the network.
PDU Protocol data unit (PDU) is a single unit of information transmitted among peer entities of a computer network.
Payload Payload contains all the data and the actual data.
CRC Cyclic Redundancy Check (CRC) : CRC is 4 Byte field. This field contains a 32-bits hash code of data, which is generated over the Destination Address, Source Address, Length, and Data field. If the checksum computed by destination is not the same as sent checksum value, data received is corrupted.
GFC Generic Flow Control : It is used to define a multiplicity of users at a common interface.

VCI VCI- The Virtual Connection Identifier is a 16-bit field used to identify paths between individual devices on the ATM network.
PT PTI-The Payload Type Indicator is a 3-bit field used to identify one of eight traffic types carried in the cell.
CLP CLP- The Cell Loss Priority bit serves the same function as the DE bit in frame relay, but identifies cells to discard when congestion occurs.
HEC The Header Error Control byte not only detects bit errors in the entire 40-bit header, but can also correct single bit errors.

Chapter 4 :IP Address
IANA IANA (Internet Assigned Numbers Authority) is the organization that oversees the allocation of IP addresses to internet service providers (ISPs).
Network ID To calculate the network id, put the network id part same and host id part 0.
Host ID A host ID can be found simply by ANDing the IP address in binary form with its respective default subnet mask (in binary form). The other fragment of an IP address is the network ID, which identifies the network to which the host belongs.
