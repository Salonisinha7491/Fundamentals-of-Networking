Access Method of Ethernet:CSMA/CD

Steps involve in CSMA/CD:-
  • Step 1:- Sense the Carrier (i.e, backbone cable).
  • Step 2:- If Carrier is idle, then transmit data.
  • Step 3:- Else if Carrier is busy, then
  • Step 4:- (a) Wait for random amount of time, It is define through algorithm i.e Binary Back off algorithm or Exponential back off algorithm
  • (b) Go to step 1

Binary BackOff Algorithm

Some Factor are :-
  1. n  = Number of collsion(for a frame or datapacket)
  2. Contension Window(C.W)  = It is a container that hold the variable,waiting time that range from 0 to 2 raise to power n-1
  3. Waiting time   = Random number choosen from Contension Window   x   T slot
  4. T slot  = Maxmium Propagation delay
  5. T slot  = 2t   where t is From Source to destination time required.

Drawback : Capture Effect

If a station win collision one time then its probability of winning successive collision is going to increase Exponentially.

To overcome this limitation of CSMA/CD, we move to the other method which is Switch Ethernet (use star topology with switch)

Summary:

  • Speed   = Upto 10 mbps
  • Access Method   = CSMA/CD
  • Topology   = BUS
  • Number of frame   = 1

ATM   (Asynchronous Transfer Mode)

It is originated in 1876 by ITU-T(International Telecommunication Union). It is a connection oriented network that support voice,video and data Communication.

ATM procedure:

  1. Line set up
  2. Communication Establish
  3. Line Terminate
Target of ATM
  1. Speed
  2. Backward compatibility
  3. Mass population
